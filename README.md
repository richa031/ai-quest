# AI Quest - Your Journey to AI Mastery

A gamified, single-page learning tracker that takes you from Python basics to building AI agents — one quest at a time.

## Live Site

**https://richa031.github.io/ai-quest/**

## What is this?

A free, static website that turns learning AI/ML into an RPG-style game. No backend, no login, no cost.

### Features
- 11 levels covering Python → Math → ML → Deep Learning → NLP → CV → GenAI → Prompt Engineering → RAG → Agents → MLOps
- 40+ quests with 100+ curated free resource links (YouTube, Coursera, Kaggle, docs)
- XP & level system with themed titles (Curious Observer → AI Overlord)
- Daily streak tracking
- Confetti celebrations on quest completion
- Level-up modals
- Progress saved locally via localStorage
- Mobile responsive — works on phone and desktop
- Dark RPG-themed UI

### Resources Include
- Andrew Ng (ML & Deep Learning Specializations)
- Andrej Karpathy (Neural Nets: Zero to Hero, Let's Build GPT)
- 3Blue1Brown (Linear Algebra, Calculus, Neural Networks)
- StatQuest (Statistics, ML algorithms)
- Hugging Face (NLP, Diffusion, Transformers)
- FreeCodeCamp, Kaggle, fast.ai, and more

## Tech Stack

- Single HTML file (~1400 lines)
- Vanilla JS + CSS (zero dependencies, zero build tools)
- Hosted on GitHub Pages (free)
- localStorage for progress persistence

## Usage

Just open the live link and start checking off quests. Progress is stored in your browser — separate per device (no sync between phone/laptop).

## Local Development

```bash
# Just open the file in a browser
open index.html
```

## Reset Progress

Click the "Reset All Progress" button at the bottom of the page, or clear localStorage for the site in your browser settings.
